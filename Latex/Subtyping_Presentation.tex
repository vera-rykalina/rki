\begin{frame}{Introduction}
\begin{block}{\textbf{HIV-1 Groups}}
Human immunodeficiency virus (HIV) displays an extraordinary genetic diversity with four distinct groups: 
\begin{itemize}
\item M (major)
\item O (outlier)
\item N (non M/O)
\item P (putative)
\end{itemize}
\end{block}  
\end{frame}
